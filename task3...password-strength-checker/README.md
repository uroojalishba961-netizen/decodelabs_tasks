# Password Strength Checker

A simple, dependency-free password strength checker built with vanilla HTML, CSS, and JavaScript — Project 3 (Interactive Web Elements) for DecodeLabs Frontend Development training.

## Features

- Live strength meter that updates as you type
- Checklist for length, uppercase, number, and special character rules
- Show/Hide password toggle
- Pure DOM manipulation — no frameworks or libraries

## How to run

Just open `index.html` in any browser. No build step, no server required.

## Files

- `index.html` — page structure
- `style.css` — styling
- `script.js` — interactivity and logic
